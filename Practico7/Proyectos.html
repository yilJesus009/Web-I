Proyecto 1: Sistema de Gestión de una Veterinaria

Tecnologías Utilizadas

  • Java
  • PostgreSQL

Funcionalidades

  • Gestión de pacientes
  • Control de citas
  • Historial médico
  • Gestión de usuarios
  • Gestión de vacunas

Este sistema fue desarrollado en Java con base de datos en PostgreSQL y está orientado a optimizar la gestión de una clínica veterinaria.

Cuenta con tres vistas principales:

  • Recepción: Permite registrar nuevos pacientes (cachorros) y sus dueños, así como agendar citas para consultas veterinarias. Ofrece una interfaz sencilla para buscar y visualizar la información de los pacientes y sus dueños.
  • Veterinario: brinda acceso al historial clínico del paciente, así como la posibilidad de aplicar vacunas (primera y segunda dosis), seleccionando el tipo de vacuna según la edad del cachorro.
  • Administrador (superusuario): permite la creación, edición y gestión de usuarios dentro del sistema.

La interfaz está diseñada para ser intuitiva y de fácil adopción. El proyecto fue desarrollado como parte de la carrera de Ingeniería en Sistemas en la Universidad NUR.

Proyecto 2: Brazo Robótico

Tecnologías Utilizadas

  • Arduino (programación y control)
  • Servomotores (4 unidades)
  • Cables macho y hembra
  • Placa Arduino
  • Placa de conexión para cableado
  • Potenciómetros
  • Botones (2 unidades)
  • LEDs (rojo y verde)

Descripción del Proyecto

Este proyecto consiste en un brazo robótico que simula una mano de producción, el cual puede ser controlado de forma manual mediante potenciómetros y también cuenta con un modo de grabación y reproducción de movimientos.

Funcionamiento

  • Control manual: los potenciómetros permiten mover cada articulación del brazo robótico en tiempo real.
  • Modo grabación:
    • Al presionar el botón de grabar, se enciende un LED verde.
    • Durante este tiempo, el usuario puede mover el brazo libremente y los movimientos quedan registrados.
  • Modo reproducción:
    • Al presionar el botón de detener grabación, el LED verde se apaga y se enciende el LED rojo.
    • En este modo, el brazo comienza a repetir todos los movimientos previamente registrados.

El sistema fue desarrollado con Arduino como plataforma principal, para un proyecto de la Universidad NUR en la carrera de Ingeniería en Sistemas.

Proyecto 3: Ajedrez

Tecnologías Utilizadas

  • Python
  • Librerías de Python para lógica de ajedrez e IA (ChessIA, ChessEngine)
  • Interfaz gráfica (Pygame)

Descripción del Proyecto

El proyecto consiste en un juego de ajedrez interactivo en el cual un jugador humano puede enfrentarse contra la computadora. El sistema implementa una IA básica que analiza los movimientos posibles y responde estratégicamente, simulando una partida real contra un oponente virtual.

Funcionamiento

  • Modo jugador vs PC: el usuario controla las piezas blancas o negras mientras la computadora responde automáticamente con movimientos calculados.
  • Reglas del ajedrez: se respetan las reglas estándar del juego, incluyendo movimientos especiales como el enroque y captura al paso de peones.
  • Interfaz gráfica: el tablero y las piezas se muestran visualmente para facilitar la interacción del jugador.
  • Lógica de IA: la computadora analiza los movimientos posibles y selecciona el que considere más favorable.

Este juego fue desarrollado como parte de la práctica de programación en Python en la materia de programacion IV, y fue presentado como proyecto final en la Universidad NUR.
